## About

Queen's Startup Summit (QSS) is a student-run organization founded in 2013 in partnership with the Queen's Innovation Connector. We support student entrepreneurship through competitions, workshops, and our flagship annual Summit — bringing together 600+ participants from 25+ schools across Canada.

## Highlights

- **600+** total participants
- **$35K+** awarded in prizes
- **200+** ideas pitched
- **25+** schools represented

## Tech Stack

[React 18](https://react.dev/) · [TypeScript](https://www.typescriptlang.org/) · [Vite 5](https://vitejs.dev/) · [Tailwind CSS 4](https://tailwindcss.com/) · [Three.js](https://threejs.org/) · [React Three Fiber](https://docs.pmnd.rs/react-three-fiber)

## Getting Started

```bash
git clone https://github.com/QueensStartupSummit/website.git
cd website
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

> **Requirements:** Node.js 18+, npm or yarn

## Scripts

| Command | Description |
| --- | --- |
| `npm run dev` | Start development server |
| `npm run build` | Production build |
| `npm run preview` | Preview production build |
